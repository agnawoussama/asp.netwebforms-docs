# Right to Left Align

TreeView supports right to left align and it can be applied by specifying **EnableRTL** as true.
    
    {% highlight html %}
    
        <ej:TreeView
            ID="treeView"
            runat="server"
            DataTextField="Text"
            DataIdField="Id"
            DataParentIdField="Parent"
            DataExpandedField="Expanded"
            EnableRTL="true">
        </ej:TreeView>
        
    {% endhighlight %}
